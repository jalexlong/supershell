# My Game World

The game takes place inside `[[The System]]`, a massive, forgotten network. The player is a `[[User]]` who gains access.

## Core Concepts
* **Mechanics:** The game is a `[[Bash Tutorial]]`. `[[Bash Commands]]` are the "magic."

* **World:** An open-world feel. The world is made of `[[Systems]]` `[[Servers]]` and `[[Networks]]`

* **Inspirations:** `[[Aesthetic - The Matrix]]`, `[[Aesthetic - Neuromancer]]`, `[[Aesthetic - Hackers]]`

* **Characters:** `[[Faction - The Admins]]`, `[[Faction - Rogue AIs]]`, `[[Faction - Other Users]]`
